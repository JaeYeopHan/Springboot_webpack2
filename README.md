# SpringBoot project scaffolding with Webpack2

## Intoduction
* SpringBoot
  * Build Tool: Gradle
  * Template Engine: handlebars

* Webpack2
  * ES6 with babel
  
* Other
  * Swagger
  

## Set up
```bash
$ mkdir springboot_webpack && springboot_webpack
$ git clone
$ cd 
$ npm install
```
